## 🔭 目前專案
- **互動式影像分割**：結合語言輸入，實現即時互動的影像分割，以達到即時互動的效果。目前的影像分割方法主要依賴手動選取，且由於語言模型過大和複雜，不利於邊緣設備的應用。本研究提出使用較小的語言模型，使邊緣設備能夠高效執行該任務。
